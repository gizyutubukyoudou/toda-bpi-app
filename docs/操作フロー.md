# 火気使用届システム 操作フロー

## 役職一覧

| 役職 | ロール | 説明 |
|------|--------|------|
| 協力会社（火元責任者） | contractor | 火気作業を行う会社の担当者 |
| 担当職員 | supervisor | 現場の担当職員 |
| 所長 | manager | 作業所の所長 |

---

## 全体フロー概要

```
① 申請作成・提出（協力会社）
        ↓ メール通知（担当職員・所長）
② 所長事前承認（所長）
        ↓ メール通知（協力会社）
③-1 現地確認（協力会社）
        ↓ メール通知（担当職員）
③-2 担当職員直前承認（担当職員）
        ↓ メール通知（協力会社）
④-1 作業中点検サイン（協力会社）
        ↓ メール通知（担当職員）
④-2 担当職員点検サイン（担当職員）
        ↓ メール通知（協力会社）
⑤ 作業終了報告（協力会社）
        ↓ メール通知（担当職員）
⑥ 残火確認（担当職員）
        ↓ メール通知（所長）
⑦ 最終承認（所長）
        ↓ メール通知（協力会社）
✅ 承認完了
```

---

## 役職別 操作手順

---

### 協力会社（火元責任者）

#### ① 申請作成・提出

1. `https://toda-bpi-app.vercel.app` にアクセスしてログイン
2. ダッシュボードの **「新規申請」** をクリック
3. 以下を入力して **「下書き保存」** → **「提出」**

| 入力項目 | 内容 |
|----------|------|
| 作業所名 | 現場の作業所名（担当職員・所長と一致させること） |
| 使用日 | 火気作業を行う日付 |
| 作業時間 | 開始・終了時刻 |
| 作業場所 | 具体的な作業箇所 |
| 火元責任者名 | 自分の氏名 |
| 作業員名 | 作業員の氏名 |
| 監視員 | 監視員の氏名・会社名 |
| 火気作業の種類 | 該当する作業にチェック |
| 可燃物・作業環境・防火対策・消火設備 | 該当項目にチェック |

4. 提出すると担当職員・所長にメール通知が届く

---

#### ③-1 現地確認（所長事前承認後）

- 所長承認のメールが届いたらアプリにログイン
- 申請詳細 → **「現地確認へ」** をクリック
- 以下を確認してチェックを入れ **「確認完了」**

| 確認項目 |
|----------|
| 可燃物の除去 |
| 床面の防護 |
| 消火設備の準備 |
| 開口部の防護 |
| 監視員の配置 |
| 火気作業表示の設置 |

---

#### ④-1 作業中点検サイン（担当職員直前承認後）

- 担当職員承認のメールが届いたらアプリにログイン
- 申請詳細 → **「作業中点検へ」** をクリック
- 作業開始時刻・残火確認時刻を入力してサイン

---

#### ⑤ 作業終了報告

- 作業が終了したらアプリにログイン
- 申請詳細 → **「作業終了報告」** をクリック
- 作業終了時刻を入力して **「報告する」**
- 担当職員に残火確認依頼メールが届く

---

### 所長（manager）

#### ② 所長事前承認

**方法A（メールから直接承認）**
1. 「新規申請が届きました」のメールを開く
2. **「事前承認する →」** ボタンをクリック
3. ブラウザに「事前承認が完了しました」と表示されれば完了

**方法B（アプリから承認）**
1. メールの **「詳細を確認する →」** をクリックしてログイン
2. 申請内容を確認して **「事前承認」** をクリック
3. 指示事項を入力（デフォルト文あり）して **「承認」**

---

#### ⑦ 最終承認

1. 「残火確認完了」のメールが届いたらアプリにログイン
2. 申請詳細 → **「最終承認」** をクリック
3. **「承認」** → 協力会社に完了メールが届く

---

### 担当職員（supervisor）

#### ③-2 担当職員直前承認（現地確認後）

1. 「現地確認完了」のメールが届いたらアプリにログイン
2. 申請詳細 → **「直前承認」** をクリック
3. **「承認」** → 協力会社に作業開始可能のメールが届く

---

#### ④-2 作業中点検サイン

1. 「作業中点検サインをお願いします」のメールが届いたらアプリにログイン
2. 申請詳細 → **「担当職員点検サイン」** をクリック
3. 点検時刻・残火確認時刻を入力してサイン → 協力会社に通知

---

#### ⑥ 残火確認

1. 「作業終了 — 残火確認をお願いします」のメールが届いたらアプリにログイン
2. 申請詳細 → **「残火確認へ」** をクリック
3. 残火確認時刻を入力して **「確認完了」** → 所長に最終承認依頼メールが届く

---

### 差し戻し（担当職員・所長共通）

- 申請内容に問題がある場合は各承認画面から **「差し戻し」** を選択
- コメントを入力して差し戻すと協力会社にメール通知
- 協力会社は内容を修正して再提出できる

---

## ステータス一覧

| ステータス | 意味 | 次のアクション担当 |
|-----------|------|-----------------|
| 下書き | 作成中 | 協力会社 |
| 所長事前確認待ち | 提出済み | 所長 |
| 現地確認待ち | 所長承認済み | 協力会社 |
| 担当職員直前承認待ち | 現地確認済み | 担当職員 |
| 作業中点検待ち | 直前承認済み | 協力会社 |
| 担当職員点検待ち | 火元責任者サイン済み | 担当職員 |
| 作業終了報告待ち | 担当職員サイン済み | 協力会社 |
| 残火確認待ち | 終了報告済み | 担当職員 |
| 最終承認待ち | 残火確認済み | 所長 |
| 承認済み | 完了 | — |
| 差し戻し | 要修正 | 協力会社 |
